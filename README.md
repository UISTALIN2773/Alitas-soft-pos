# Alitas-soft-pos
Alitas-soft-pos es un sistema POS de escritorio para restaurantes de comida rápida y alitas. Optimiza ventas, control de caja y gestión de menú con una interfaz moderna en Python y SQLite. Ofrece reportes financieros precisos, soporte para impresoras térmicas y gestión de imágenes local, garantizando un servicio ágil sin depender de internet.
